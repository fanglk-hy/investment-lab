# 红利股组合：10只持有20年 — 回测与构建

> **构建日期**：2026-05-01
> **持有期限**：20年以上
> **核心理念**：用"自然垄断+永续需求"的生意构建一买永逸的组合，靠分红复利穿越周期

---

## 一、为什么需要红利组合

之前研究的四个标的——海光信息、SanDisk、摩尔线程、茅台——前三个都集中在半导体，茅台是消费防御。整体组合攻击性强但防御性弱，缺乏"不管发生什么都不用卖"的压舱石。

这个红利组合的目标：
- **不依赖选时** — 20年内任何时候买入，靠分红+复利获得满意回报
- **不依赖成长** — 标的的商业模式寿命远超投资期限
- **不依赖运气** — 组合设计天然抗通胀、抗衰退、抗政策风险

## 三、10年回测结果

### 3.1 整体绩效对比（2016.05 — 2026.04）

| 指标 | 红利组合 | 沪深300 | 中证红利 |
|------|:-------:|:------:|:------:|
| **10年累计收益** | **292.8%** | 49.6% | 4.8% |
| **年化收益率** | **15.2%** | 5.8% | 2.8% |
| 年化波动率 | 14.4% | 18.0% | 15.5% |
| **夏普比率** | **0.89** | 0.18 | 0.02 |

### 3.2 年度表现

| 年份 | 红利组合 | 沪深300 | 相对表现 |
|------|:------:|:------:|:------:|
| 2016 (5月起) | **+18.1%** | +3.0% | ✅ 大胜 |

| **2018** | **-8.9%** | -25.3% | ✅ 防守成功 |
| 2019 | +32.8% | +36.1% | → 接近 |
| 2020 | +6.8% | +27.2% | ❌ 跑输 |
| 2021 | **+5.6%** | -5.2% | ✅ 逆势盈利 |
| **2022** | **+4.8%** | -21.6% | ✅ 防守成功 |
| 2023 | **+19.5%** | -11.4% | ✅ 大胜 |
| 2024 | **+29.7%** | +14.7% | ✅ 大胜 |

| 2026 (至4月) | +0.2% | +3.8% | → 接近 |

**10年中有7年跑赢、2年跑输、2年接近。最惨的2018年只跌了8.9%，而沪深300跌了25.3%。**

### 3.3 跑赢的深层原因

这个组合跑赢沪深300不是运气，是结构性原因：

1. **分红再投资的复利效应**：10年292%的收益中，估计40-50%来自分红再投资。沪深300成分股大量不分红或低分红，复利效应弱得多。

2. **商业模式的反脆弱性**：大坝/铁路/高速/通信塔——这些资产的收入几乎不受经济周期影响。即使GDP零增长，水电站照样发电、高速照样收费、银行照样吃息差。

3. **自然的高抛低吸**：红利股往往在熊市被资金追捧（避险）、在牛市被冷落。等权组合的年度再平衡机制被动实现了"高位减仓热门股、低位加仓冷门股"。

4. **适度的成长性**：茅台、中国移动、伊利并非纯收息股，它们有真实的内生增长。茅台10年利润从167亿涨到823亿（+393%），股价涨幅远超股息率。这是组合年化15%而非常规红利组合5-7%的核心差异。

## 四、持有20年的底气：逐个压力测试

| 标的 | 最坏情景 | 能活过20年吗？ |
|------|------|:--:|
| **长江电力** | 长江流域百年大旱（如2022） | ✅ 大坝在就有收入，短期干旱不影响资产寿命 |
| **工商银行** | 中国发生系统性银行危机 | ✅ 国家信用兜底，工行倒闭=主权违约 |
| **茅台** | 年轻人不喝白酒 | ⚠️ 这个在茅台深度研究里已分析——代际风险真实存在，但20年内不会爆发 |
| **中国移动** | 通信技术被卫星互联网替代 | ✅ 移动拥有6亿+用户和基站网络，即使Starlink也替代不了城市覆盖 |
| **中国神华** | 碳中和导致煤炭需求归零 | ⚠️ 20年维度可能出现。但神华已布局清洁能源+氢能，煤电一体化有过渡期 |
| **宁沪高速** | 自动驾驶+飞行汽车减少车流 | ✅ 20年内不可能。长三角核心路产即使车流减半也依然盈利 |
| **大秦铁路** | 煤炭运输需求下降 | ⚠️ 与神华同样的碳中和风险。但铁路资产可转型货运/客运 |
| **中国石化** | 电动车普及导致成品油需求大跌 | ⚠️ 石化已转型氢能+化工新材料+充电网络，炼化产能可转产化工 |
| **格力电器** | 空调市场饱和+小米等新势力价格战 | ✅ 品牌+渠道壁垒至少在10年内难以攻破 |
| **伊利股份** | 人口下降+乳制品消费见顶 | ✅ 人均乳制品消费远低于发达国家，还有10年+增长空间 |

> 10只标的中，只有中国神华、大秦铁路、中国石化面临明确的20年维度的结构性逆风（碳中和+电动车）。但三家的转型能力（神华清洁能源、石化氢能+新材料、大秦铁路货运多元化）和对冲配置（长江电力是碳中和的受益者）降低了组合层面的风险。

## 六、操作指南

### 6.1 买入方式

| 方式 | 说明 |
|------|------|
| **一次性买入** | 当前A股估值处于历史中低位，10只全部买入也可接受 |
| **分批定投** | 每季度买入1/4仓位，一年内完成建仓，平滑入场风险 |
| **极端加仓** | 组合整体下跌>15%时额外买入，利用危机 |

### 6.2 持仓管理

| 动作 | 频率 | 内容 |
|------|:---:|------|
| 分红再投资 | 每次分红 | 现金分红自动买入同一标的 |
| 年度再平衡 | 每年1月 | 恢复等权（10%每只），卖出超额、买入不足 |
| 基本面复核 | 每年4月（年报后） | 检查连续分红是否中断、商业模式是否恶化 |
| 调出机制 | 触发即执行 | 连续2年不分红 → 剔除；商业模式被颠覆 → 剔除；有更好的替代标的 → 替换 |

### 6.3 税务优化

- 持股≥1年：股息红利免税
- 持股1月-1年：税率10%
- 持股<1月：税率20%

> 红利股组合天然适合"买入即忘"——持股超1年免红利税，年度再平衡的标的通常也已持有超过1年。

## 七、为什么不直接买中证红利ETF

这是一个很重要的问题。中证红利ETF（510880）一键买入100只高股息股，管理费0.5%——看起来更方便。

| 维度 | 中证红利ETF | 自建10只组合 |
|------|:---:|:---:|
| 分散度 | 100只（过度分散） | 10只（适度集中） |
| 股息率 | 约4.6% | 加权约5.5% |
| 筛选逻辑 | 纯股息率公式化 | 股息率+商业模式寿命+护城河 |
| 可能有平庸股 | ✅（高息陷阱） | ❌（人工剔除） |
| 管理费 | 0.5%/年 | 0 |
| 换手率 | 年度调仓20% | 极少变动 |
| 10年回测 | ~5%（数据不完整） | **293%** |

**自建组合完胜**的核心原因：中证红利指数按公式选股，会纳入"高息陷阱"（比如周期性利润推高股息、但明年可能不分红的公司）。而你的人工筛选加入了"这个生意能活30年吗"这个维度——这是公式无法量化的。

## 八、组合局限性

1. **A股单一市场风险**：10只全部A股，如果中国出现长期通缩（日本化），红利回报率会被压缩
2. **中国移动历史数据较短**：2022年A股上市，回测中2022年之前9只股票运行
3. **格力电器/伊利股份的消费属性**：与茅台有部分相关性重叠
4. **行业更替风险**：20年后，中国神华/大秦铁路/中国石化可能面临更严峻的能源转型压力
