## 一、投资主题：2026 = AI 智能体元年

### 1.1 为什么是现在？

| 指标 | 2023-2025（训练时代） | 2026-2028（智能体时代） |
|------|----------------------|------------------------|
| AI工作负载 | 训练占80% | **推理占70-80%** |
| 芯片需求 | GPU一家独大 | GPU + ASIC + 边缘NPU 三轨并行 |
| 软件模式 | 按用户收费(SaaS) | **按Agent产出收费**(Outcome-based) |
| 企业采用 | 实验性/R&D | **40%应用嵌入Agent**(2026年末) |
| 市场增速 | AI训练市场CAGR 50%+ | **AI Agent市场CAGR 45%+** |

### 1.2 核心判断

AI正在从"被动的聊天工具"进化为**能自主执行多步任务的智能体**。这一转变的产业链影响超过从2G到4G的跃迁——因为它不仅创造新需求，还在重新定义软件、算力、网络的定价模型。

**智能体时代最受益的五个环节**：
1. **算力基础设施**（GPU/ASIC/网络）— Agent数量指数增长直接拉动芯片和带宽需求
2. **存储与内存**（HBM/DRAM/NAND）— Agent的上下文窗口从4K扩展到百万token级别
3. **AI Agent平台**（企业软件嵌入Agent）— 软件从"按人头收费"到"按Agent产出收费"
4. **光互连与网络**（800G/1.6T/CPO）— 带宽是推理时代的最大物理瓶颈
5. **边缘AI与终端**（NPU/AIoT/机器人）— Agent从云端"下沉"到设备端

### 1.3 组合定位

| 维度 | 参数 |
|------|------|
| 持有时长 | 2-3年，不做短期波段 |
| 基准指数 | S&P 500 (SPY) / 沪深300 (000300) |
| 目标超额收益 | 年化跑赢基准 **15-25%+** |
| 最大回撤容忍 | 15-25%（高于整体组合的10-20%） |
| 再平衡 | 每半年 |
| 退出条件 | AI CapEx增速连续两季 <10%；或组合整体PE>35x |

## 七、组合管理纪律

### 7.1 建仓策略

| 方式 | 说明 |
|------|------|
| **分批建仓** | 首批50%在当前价位，30%在首次回调-10%时加仓，20%保留现金管理 |
| **单票上限** | 8%（接近但不超过你10%的上限） |
| **单次建仓量** | 不超日均成交量的5% |

### 7.2 退出条件（按纪律执行）

1. **基本面退出**：AI相关CapEx增速连续两个季度 <10% → 组合整体减仓50%+
2. **估值退出**：组合整体PE >35x时减仓至60%，PE >45x时空仓
3. **个股退出**：单个标的连续两季营收miss且下调指引 → 立即清仓
4. **机会退出**：发现更优标的时，替换而非叠加

### 7.3 再平衡规则

- **每半年**（5月/11月）：恢复目标权重
- **阈值触发**：单票偏离目标权重±30%时即时再平衡
- **动量保留**：允许盈利标的自然超配（不强制削减赢家）
