## 二、中美利差与汇率

### 2.1 当前利差结构

```
美国10Y: 4.37%
中国10Y: 1.74%
─────────────────
利差倒挂: -263bp (深度倒挂)
```

这是近20年来最深的中美利差倒挂之一。正常情况下，这种利差应该导致人民币大幅贬值——但实际没有。

### 2.2 为什么人民币反而走强？

| 因素 | 影响 |
|------|------|
| 中国经常账户顺差（2025年约$5,000亿+） | 基本面支撑 |
| 全球"去美元化"央行购金/购人民币 | 结构性资金流入 |
| 中国资本市场估值低位（A股PE处于历史低位） | 外资边际增配 |
| 美国财政赤字/GDP突破7% | 美元信用侵蚀 |

> 人民币从2025年底的7.0+升值至6.83，升值幅度约2.5%。这种"利差不贬反升"意味着市场在定价某种更深层的变化——可能是美元信用的长期弱化。

### 2.3 组合影响

| 资产 | 汇率影响 |
|------|---------|
| **A股（茅台、海光）** | 人民币升值利好外资流入，但幅度有限 |
| **美股（SanDisk）** | 人民币升值→换汇成本下降→买入美股更便宜 |
| **港股** | 人民币升值+港股低估值=双击机会 |

## 三、关税与出口管制

### 3.1 最新升级：MATCH 法案

2026年4月22日，美国众议院通过《硬件技术多边协调管制法案》（MATCH Act），这是继2022年10月7日规则后最严厉的芯片出口管制：

**关键条款**：
- 点名中国五大科技企业（华为、中芯国际、长江存储、长鑫存储、华虹）全面封锁
- 卡死28nm成熟制程——不限于AI芯片，连汽车/家电用的"口粮"芯片都禁
- DUV浸没式光刻机全面禁售，取消原有许可证
- 禁止ASML为中国现有设备提供售后服务（维修、零配件）
- 胁迫荷兰/日本150天内对齐美国标准

### 3.2 华虹断供事件

2026年4月29日，美国商务部向应用材料、泛林、科磊发出通知，暂停向华虹半导体（中国第二大代工厂）供应设备。美方认为华力微电子已具备7nm能力。

### 3.3 中方的反制与对冲

| 日期 | 动作 |
|------|------|
| 4月25日 | 商务部：未经允许，核心科技企业不得接受美资 |
| 4月27日 | 否决Meta收购Manus AI案（$20亿） |
| 持续 | 上海微电子DUV光刻机支撑28nm量产 |
| 持续 | 长鑫存储HBM2e量产、HBM3年内 |
| 持续 | 华为+百度/阿里部署全自主万卡AI集群 |

### 3.4 "习川会"前夕的信号博弈

特朗普计划5月中旬访华。当前密集施压（4月22日通过22项对华法案）既是筹码积累，也说明双方在谈判桌上仍有交易空间。

### 3.5 组合影响（直接影响现有持仓）

| 标的 | 制裁影响 | 强度 |
|------|---------|:----:|
| **海光信息（688041）** | **正面** — 制裁升级=国产替代逻辑强化，DCU需求暴增 | ⭐⭐⭐⭐⭐ |
| **SanDisk（SNDK）** | 中性偏负 — 日本制造/合资结构暂时豁免，但技术出口管制扩散风险 | ⭐⭐ |
| **茅台（600519）** | 中性 — 与科技制裁无关 | ０ |
| **半导体ETF** | A股正面（国产替代）/ 美股负面（出口受限） | ⭐⭐⭐ |

> 核心判断：制裁利好的不是所有中国芯片公司，而是**能替代进口的**。海光的x86 CPU和DCU是直接替代Intel/NVIDIA的，制裁越狠，替代空间越大。但如果制裁进一步扩散到日本（限制台积电/铠侠为中国设计企业代工），海光的代工渠道也会受冲击。

## 四、全球流动性

### 4.1 美联储：鹰派降息预期退潮

| 时间 | 事件 |
|------|------|
| 4月29日 | FOMC维持3.50-3.75%，4票反对（1992年以来最大分歧） |
| 5月15日 | 鲍威尔卸任，凯文·沃什正式接任主席 |
| 当前定价 | 年内降息概率仅15-22%，最早2027年4月才有50%概率 |
| 沃什立场 | 鹰派、通胀优先、"缩表与降息可并行" |

**当前困境**：油价因美伊冲突飙至$114-126/bbl → 通胀预期回升 → 不能降息 → 但高利率侵蚀财政可持续性（美国债利息已超军费）

### 4.2 美联储缩表

| 指标 | 数据 |
|------|------|
| 资产负债表规模 | ~6.7万亿美元（较峰值~9万亿缩减超2万亿） |
| 每月购债缩减 | 400亿→250亿美元 |
| 总缩表目标（前官员Miran） | 可能需缩减1-2万亿（需要数年） |

> 缩表就是抽走流动性。历史上每次缩表后期都出现流动性危机（2019年9月回购利率飙升）。当前缩表速度加快，准备金/银行资产比率逼近"稀缺"阈值。一旦爆发美元流动性危机，所有风险资产都会同步下跌——这是茅台和海光唯一会同时暴跌的情景。

### 4.3 中国流动性

| 指标 | 数值 | 评价 |
|------|------|------|
| M2增速 | +8.5% | 适度宽松 |
| 社融Q1增量 | 14.83万亿 | 历史高位 |
| M1-M2剪刀差 | 低于5pct（近3年最低） | 企业活力改善 |
| 企业债券融资 | 同比+3.6pct | 直接融资结构优化 |

流动性充裕，但问题是钱在银行体系内空转，没有大规模流入股市。A股成交量中枢下移，市场缺乏增量资金。

### 4.4 黄金：关键信号

- 1月29日历史高点 $5,595 → 当前 $4,602（-17.7%）
- 央行Q1购金244吨（+3%），Q1总需求$1,930亿（历史纪录）
- 油价飙涨→通胀→实际利率上行→压制黄金（短期承压）
- 但去美元化趋势未变：非美央行黄金储备总值已超海外持有美债总额

黄金回调17%意味着什么？**有两种解释**：
1. 流动性危机中黄金也被抛售（类似2020年3月）→ 利空信号
2. 央行购金需求提供底部支撑，回调是长期配置窗口 → 利多信号

我的判断：更接近第二种。央行购金是结构性需求，与投机性ETF流出不同。黄金在$4,200-4,500区间是战略配置窗口。

## 六、组合层面情景推演

基于以上宏观变量，梳理对当前研究标的的三大情景：

### 情景一：基准情景（概率 ~55%）— 现状延续

| 假设 | 指标 |
|------|------|
| 美联储 | 利率维持3.50-3.75%至年底 |
| 中美关系 | 习川会达成局部协议，制裁不进一步升级但也不放松 |
| 油价 | $100-120区间震荡 |
| 人民币 | 6.75-6.95区间 |
| 中国经济 | L型底部震荡，无强刺激 |

| 标的 | 预期表现 | 12个月预期回报 |
|------|---------|:------------:|
| 海光信息 | 国产替代逻辑持续，营收+50%以上 | +15-30% |
| SanDisk | NAND价格温和回落，长协执行平稳 | -5% ~ +10% |
| 茅台 | Q2-Q4增速改善，估值温和修复 | +5-15% |
| 组合（等权） | — | +5-18% |

### 情景二：风险偏好改善（概率 ~20%）— 中美缓和 + 油价见顶

| 触发条件 | 习川会达成实质性协议、美伊停火、油价回落至$80 |
|----------|------|

| 标的 | 预期表现 | 逻辑 |
|------|:--------:|------|
| 海光信息 | -10%~0% | 制裁缓和=国产替代紧迫性下降，短期承压 |
| SanDisk | +20-40% | 风险偏好回升+AI算力需求驱动 |
| 茅台 | +15-25% | 经济预期改善→消费复苏→估值修复 |
| 组合（等权） | **+8-22%** | |

### 情景三：尾部风险（概率 ~25%）— 中东升级 + 美元流动性危机

| 触发条件 | 美伊开战、油价冲$150+、霍尔木兹海峡长期封锁、美元荒 |
|----------|------|

| 标的 | 预期表现 | 逻辑 |
|------|:--------:|------|
| 海光信息 | -30~-50% | 流动性危机中成长股首当其冲，台积电断供风险 |
| SanDisk | -50~-70% | NAND商品+高估值+周期见顶，三重打击 |
| 茅台 | -10~-20% | 防御属性发挥作用，但流动性危机中无资产幸免 |
| 组合（等权） | **-30~-47%** | |

> **这个尾部情景值得注意**：在美元流动性危机中，茅台和海光的相关性会从平时的~0.1飙升至0.6-0.7。所有资产都会被同步抛售（除美元和短期美债）。组合的"防御性"在这种情况下基本失效——唯一有效的对冲工具是黄金和现金。

### 情景概率加权回报

| 情景 | 概率 | 组合加权回报 |
|------|:---:|:-----------:|
| 基准 | 55% | +11.5% |
| 改善 | 20% | +3.0% |
| 尾部 | 25% | -9.6% |
| **概率加权年预期回报** | **100%** | **+4.9%** |

> 在当前宏观环境下，组合的年化预期回报约5%。不算高——因为25%的尾部风险概率拖累了整体期望。这个数字提醒我们：**现金和黄金的配置比例不能太低。**
