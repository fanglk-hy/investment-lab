### 1.2 产业链位置与本次研究的覆盖

```
AI训练/推理需求
  ├─ [GPU/加速器]  → 英伟达 (NVDA)          ← AI 算力核心
  ├─ [晶圆代工]    → 台积电 (TSM)            ← 制造垄断者
  │                → 中芯国际 (SMIC)          ← 中国替代
  │                → 华虹半导体 (Hua Hong)    ← 特色工艺
  ├─ [存储芯片]    → 三星电子 (Samsung)      ← DRAM/NAND 巨头
  │                → SK海力士 (SK Hynix)     ← HBM 龙头
  │                → 美光科技 (Micron)        ← HBM 第三极
  ├─ [光互连]      → 中际旭创 (300308)       ← 800G/1.6T 光模块
  └─ [云计算平台]  → 谷歌 (Alphabet/GOOGL)   ← AI 需求方+TPU自研
```

### 1.3 核心判断：超级周期处于早期偏中段

- **需求端**：AI训练→推理→Agent生态的三级火箭，存储/算力/互连需求远未见顶
- **供给端**：先进制程（3nm以下）+ HBM + 先进封装三重产能瓶颈，2027年前难以缓解
- **价格端**：DRAM合约价Q1环比+83-95%，NAND合约价Q1环比+55-160%，Q2继续走强
- **风险信号**：CapEx激增（TSMC $52-56B, 三星$73B, 美光$25B+），历史上CapEx高峰常领先周期拐点12-18个月

## 二、英伟达 (NVIDIA, NVDA) — AI 时代的"卖铲人"

### 2.1 商业模式

| 维度 | 描述 |
|------|------|
| **核心业务** | GPU + 加速计算平台（Blackwell/GB300 → Vera Rubin → Rubin Ultra） |
| **收入结构(FY2026)** | 数据中心 ~$240B (88%), 游戏 ~$12B, 专业可视化 ~$3B, 汽车 ~$2B |
| **毛利率** | Q4 FY2026 75.2%, Q1 FY2027 指引 ~74.9% |
| **客户集中度** | 前4大客户(SMCI/Dell/微软/Meta等)占收入 ~45% |
| **定价模式** | GPU整卡+NVLink+InfiniBand/Spectrum-X交换机+软件(CUDA)体系绑定销售 |

**现金流质量检查**：
- FY2026经营现金流 ~$100B+, 净利润 ~$120B, 经营现金流/净利润 ≈ 0.83
- 自由现金流 FY2026 ~$75B+, FCF利润率 ~30%+（剔除CapEx后依然优秀）
- 应收账款增速合理，库存在Blackwell过渡期略有上升但周转良好

### 2.2 护城河

| 维度 | 评级(1-10) | 说明 |
|------|:---:|------|
| **CUDA生态锁定** | 10 | 500万+开发者、300+SDK、10年+积累，网络效应极强 |
| **硬件性能领先** | 9 | Blackwell → Rubin路线图清晰，节奏12-18个月一代 |
| **系统级整合** | 9 | GPU + NVLink + NVSwitch + DPU + 交换机 = 完整数据中心方案 |
| **规模效应** | 8 | 毛利率75%+，对手(AMD)仅50%+，价格战空间巨大 |

**护城河趋势：变宽，但面临结构性挑战**
- **变宽因素**：CUDA生态锁定随AI应用扩散而强化；Rubin架构每瓦性能提升3x
- **变窄风险**：超大规模客户(CSP)自研ASIC（Google TPU v6、Amazon Trainium3、微软Maia、Meta MTIA v3）——这可能是NVDA历史上最大的结构性质变

### 2.3 竞争对手分析

| 对手 | 威胁等级 | 说明 |
|------|:---:|------|
| **AMD (MI400)** | ★★★ | 性能追赶中，但软件生态(CUDA vs ROCm)差距仍在3-5年 |
| **CSP自研ASIC** | ★★★★★ | **最大长期威胁**——训练：推理比从80:20趋于50:50，ASIC在推理侧可以替代GPU |
| **Intel (Gaudi 3/Falcon Shores)** | ★★ | 份额极小，战略摇摆 |
| **Cerebras/Groq等创业公司** | ★★ | 特定场景有优势，但缺乏通用性 |

### 2.5 催化剂与风险

**催化剂**：
- Rubin Ultra Q3 2026发布，每瓦性能再提升3x
- Q1 FY2027(May 20)财报，营收指引$78B若再次beat将推动重估
- 主权AI(国家GPU采购)+企业AI渗透加速

**风险**：
- CSP自研ASIC是最大结构性威胁（Graviton→Trainium模式复制）
- H20出口管制导致~$80亿中国收入清零
- 估值虽合理但AI叙事一旦转弱，PE压缩空间巨大

### 2.6 管理层

- **黄仁勋(Jensen Huang)**: 创始人+CEO，30年+行业经验，技术远见极强
- 资本配置：$750亿+回购计划，无分红，CapEx以供应链预付款为主
- **正面**：连续12个季度beat and raise执行记录无可挑剔
- **负面信号**：2025年以来黄仁勋多次减持（计划性10b5-1，但时点在股价高位）

### 2.7 Pre-mortem：3年后这笔投资腰斩的死因

**最可能死因：CSP自研ASIC大规模替代 GPU 训练需求**

谷歌TPU v6(2027)、亚马逊Trainium3(2027)、微软Maia v3(2028)合计可替代30-40%训练市场，NVDA增长从+30%骤降至0%，PE从40x压缩至18x → 股价腰斩。

**认真审视**：这个"威胁"已说了5年，每次NVDA都用更强的产品化解。但这次不同——所有云厂商同时在推自研芯片，且推理占比上升让ASIC在50%的AI工作负载中有经济可行性。**这不是"狼来了"，而是"狼群来了"**。

### 3.2 护城河 — 半导体行业最宽的护城河之一

| 维度 | 评级(1-10) | 说明 |

### 3.4 Pre-mortem：3年后腰斩的死因

- 任何军事冲突都会导致供给中断

**更现实的死因**：三星/Intel在2nm GAA技术上反超，打破TSMC的"独家供应商"地位 → PE从30x压缩至18x。概率：10-15%。
