# Investment Lab · 投资研究实验室

全球多资产长期价值投资研究项目。

## 投资哲学

遵循查理·芒格、沃伦·巴菲特、但斌的投资理念：

- 寻找有深厚护城河的优质企业，以合理价格买入并长期持有（5 年以上）
- 运用多元思维模型，理解商业模式本质
- 与伟大企业共成长，聚焦消费 / 科技 / 能源 / 金融赛道

## 分析框架

每只股票覆盖八个维度：

1. **商业模式** — 怎么赚钱？收入结构、客户粘性
2. **护城河** — 为什么对手抢不走生意？护城河在变宽还是变窄？
3. **竞争对手** — 行业格局、份额变化
4. **行业周期** — 处于周期的什么位置？
5. **催化剂与风险** — 未来 1-3 年的关键变量
6. **估值** — 定量 DCF/PE/可比估值，给出合理区间
7. **管理层与治理** — 资本配置能力、诚信、股权结构
8. **逆向验证** — 假设3年后归零，最可能的死因是什么？
